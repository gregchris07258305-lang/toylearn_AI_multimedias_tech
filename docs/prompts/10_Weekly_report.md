# 1. 일반화 프롬포트 요청
```javascript

프로젝트 현황 데이터를 기반으로 하기의 요구사항을 반영한 '경용진 용 주간 보고서'를 작성하는 프롬포트를 만들려고합니다.

일관된 a4 용지 반페이지 분량의 아웃풋이 나올 수 있겠끔 프롬포트를 제작해주세요.

[요구사항]
 + 여러 프로젝트 데이터를 종합 분석
 + 우선순위와 리스크 자동 판별
 + 경영진이 의사결정할 수 있는 형식으로 간단히 요약
 + 일관된 보고서 형식 유지 (가장 중요)

[프로젝트 현황 데이터_예시]
프로젝트명: 신규 ERP 시스템 도입
담당팀: IT운영팀

진행률: 65%
예산 집행률: 58%

주요 이슈: 
- 외부 벤더와의 일정 조율 지연 (3일)
- 회계팀 요구사항 추가 접수

완료 작업:
- 시스템 설계 완료
- 테스트 서버 구축 완료

다음 주 계획:
- 회계 모듈 개발 착수
- 사용자 교육 자료 준비
```
